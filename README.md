# GiphyIOSTestTask

1) clone:
git clone https://github.com/rusel95/GiphyIOSTestTask.git
2) install pods (need to have installed cocoapods https://cocoapods.org/):
cd GiphyIOSTestTask
pod install
3) Open project:
go to project's folder and open GiphyIOSTestTask.xcworkspace
4) select your team:
click on GiphyIOSTestTask -> select 'General' tab -> select 'Team' (probably your personal account)
5) Run
cmd + R
